# JPT Fashion Website

A luxury fashion website inspired by YSL.com, featuring Jean Philippe Tourigny as the fashion designer.

## Features

- **Hero Section**: Auto-rotating slideshow with luxury fashion imagery
- **About Section**: Detailed presentation of Jean Philippe Tourigny
- **Collections Gallery**: 4 featured fashion collections
- **Blog Section**: Fashion-related articles and insights
- **Contact Section**: Contact information and appointment form
- **Footer**: Links and newsletter signup

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Lucide React Icons

## Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run the development server:**
   ```bash
   npm run dev
   ```

3. **Open [http://localhost:3000](http://localhost:3000) in your browser**

## Build for Production

```bash
npm run build
npm start
```

## Design Inspiration

The design is inspired by luxury fashion websites, particularly YSL.com, featuring:
- Minimalist black and white color scheme
- Elegant typography with wide letter spacing
- Smooth animations and transitions
- High-quality fashion photography
- Sophisticated layout and spacing

## Customization

You can easily replace the placeholder images from Unsplash with your own fashion photography by updating the image URLs in the component.
